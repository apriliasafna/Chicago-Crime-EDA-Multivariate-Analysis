# Chicago Crime EDA Multivariate Analysis (2015–2020)

Proyek ini berisi analisis dan visualisasi data multivariat terhadap pola kejahatan di Kota Chicago periode 2015–2020 menggunakan pendekatan Exploratory Data Analysis (EDA).

## Tujuan
- Menganalisis pola kejahatan berdasarkan waktu
- Mengidentifikasi jenis kejahatan dominan
- Menganalisis persebaran lokasi kejahatan
- Mengkaji hubungan antara lokasi dan tingkat penangkapan

## Dataset
Sumber Dataset: https://www.kaggle.com/code/redlineracer/chicago-crime-map-visualisation-and-analysis.

Dataset yang digunakan adalah Chicago Crime Dataset periode 2015–2020 yang mencakup:
- Jenis kejahatan (primary_type)
- Waktu kejadian (year, date)
- Status penangkapan (arrest)
- Lokasi (latitude & longitude)

## Visualisasi
Analisis dilakukan menggunakan beberapa teknik visualisasi:
- Heatmap
- Facet Bar Plot
- Line Plot
- Boxplot (Latitude & Longitude)
- Scatter Plot

##  Insight Utama
- Pola kejahatan berbeda pada setiap jenis kejahatan
- Robbery merupakan jenis kejahatan paling dominan
- Lokasi kejahatan cenderung terpusat dan stabil
- Tingkat penangkapan pelaku relatif rendah

- Pandas
